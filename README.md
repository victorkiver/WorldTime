#### WorldTime

A beautifully crafted iOS app for tracking time across the globe. Built entirely with SwiftUI.

#### Features

• Dashboard — Live world clock cards with UTC offsets, city names, and day/night indicators. Staggered entrance animations and a horizontally scrollable photo gallery with circle-reveal fullscreen viewer.

• Clock — Large digital clock with animated numeric transitions, location details, and daylight duration. Includes 12h/24h format toggle and a theme picker sheet with sun/moon morphing animation.

• Map — World map visualization with a horizontal card carousel of timezone-mapped cities.
